Contacts API Documentation

This API provides CRUD operations for managing contact details. It is deployed on: https://your-api-url.com

API Authentication

To access this API, you must include an API key in the request headers:

Header: x-api-key
Value: (Request API Key via email: ebinuga@hotmail.com)

Endpoints

Fetches all saved contacts.

Fetches a specific contact by ID.

Creates a new contact.

Updates an existing contact.

Deletes a contact by ID.

Note: Ensure you include the API key in the request headers to authenticate successfully.
